# BlogPrompt Tool

> AI(ChatGPT, Claude 등)에 붙여넣을 한국어 블로그 프롬프트를 생성하고, 등록 이력을 관리하는 도구

---

## 🧩 주요

### 초안 만들기
| 타입 | 설명 |
|------|------|
| 일반 주제형 | 메인/세부 주제 선택 → 한국어 블로그 프롬프트 생성 |
| 오늘의 역사형 | 날짜 기준 역사적 사건 프롬프트 생성 |

### 등록 목록
- 항목 추가 / 삭제 / 검색
- 중복 방지 기준 데이터로 활용

### 종합 통계
- 주제별 · 타입별 현황, 키워드 빈도, 최근 등록 글

## 🎨 기술 스택

| 역할 | 라이브러리 |
|------|-----------|
| UI | React 18 + TypeScript |
| 빌드 | Vite 5 |
| 스타일 | SCSS Modules |
| 상태관리 | Zustand (persist) |
| 라우터 | React Router v6 |
| 저장소 | localStorage |
